### English

This plugin extends GLPI with comprehensive technician activity management:

* Track **activities** in the GLPI planning calendar with color-coded external events by type (activity, holiday, ticket, manageentities).

* Manage **holiday requests** with a full approval workflow: technicians submit requests, validators approve or refuse, and notifications are sent at each step.

* Display **public holidays** in the planning calendar.

* Generate a monthly **Activity Report (CRA)** per technician, exportable as a PDF, aggregating activities, holidays, ticket tasks, and project tasks.

* View **statistics** of time distribution by technician, event category, or project.

* Maintain **holiday counters** (paid leave, RTT, etc.) per user.

### Français

Plugin activity pour GLPI.

Le plugin est utilisé pour la gestion des activités dans le planning, la gestion des congés, l'affichage des jours fériés, et la génération du Compte Rendu d'Activité (CRA).

* Suivre les **activités** dans le planning GLPI via des événements externes colorés par type.

* Gérer les **demandes de congés** avec un circuit de validation : les techniciens soumettent leurs demandes, les valideurs les approuvent ou les refusent.

* Afficher les **jours fériés** dans le planning.

* Générer un **Compte Rendu d'Activité (CRA)** mensuel par technicien, exportable en PDF.

* Consulter des **statistiques** de répartition du temps par technicien, catégorie ou projet.

* Gérer des **compteurs de congés** (CP, RTT, …) par utilisateur.
